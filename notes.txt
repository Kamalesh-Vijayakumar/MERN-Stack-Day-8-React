React :
React is a free and open-source front-end JavaScript library 
for building user interfaces based on UI components. It is 
maintained by Facebook and a community of individual developers
and companies. React can be used with a variety of JavaScript 
libraries and frameworks, such as Angular, Backbone, and Ember.js.
